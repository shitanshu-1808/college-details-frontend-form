# Advanced Registration Form (HTML & CSS)

## Project Overview

This project is an advanced registration form built using HTML and CSS. The structure and styling are separated into two different files to improve organization, readability, and maintainability.

The form is designed to be clean, professional, and user-friendly, focusing on simplicity and clarity without unnecessary complexity.

---

## Form Details

The registration form includes the following fields:

- Name  
- Roll Number  
- Branch  

Each field uses proper labels and placeholders to ensure clarity, accessibility, and ease of use for the user.

---

## Frontend Structure

- The HTML file contains the complete structure of the registration form
- Semantic elements, labels, and placeholders are used for better readability and accessibility
- The form layout is designed to be simple and human-friendly

---

## Styling

- A separate CSS file is used for styling the form
- Focus is placed on:
  - Clean layout
  - Proper spacing
  - Alignment
  - Professional appearance
- The design avoids unnecessary visual complexity while maintaining clarity

---

## Responsiveness

- The form is designed to be responsive
- It adapts smoothly to different screen sizes
- Ensures usability across devices while maintaining a readable and clean interface

---

## Summary

This project demonstrates good frontend practices by separating structure and styling, using accessible form elements, and creating a responsive, clean, and professional registration form using only HTML and CSS.
